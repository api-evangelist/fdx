The FDX API Version 6.4 release incorporates 24 new RFCs including YAML structure updates, a new CSDF Security Model, revised User Experience Guidelines, the Consent API Behavioral Specification, and the FDX Consensus Standard Data Format v1 tailored for CFPB compliance demonstration.
